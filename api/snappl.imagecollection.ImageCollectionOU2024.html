Return a OpenUniverse2024FITSImage based on specifications.

If you specify all of path, pointing, band, and sca, they must be consistent.

If you just give path, then pointing, band, and sca will be read from the header.

If you give pointinb, band, and sca, but not path, it will use get_image_path to find the image.
