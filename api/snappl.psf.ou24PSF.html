Don’t call this or the constructor of a subclass directly, call PSF.get_psf_object().

See documentation on the various subclass constructors for the arguments to apss to get_psf_object and what they all mean. Different types of PSF need different arguments, and sometimes the same argument names will mean different things for different subclasses!
