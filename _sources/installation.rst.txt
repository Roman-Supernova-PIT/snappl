============
Installation
============

Normally, you will use the version of snappl that is installed in the :doc:`environment`.

You can install snappl into your working environment via pip with:

.. code-block:: console

  $ pip install roman_snpit_snappl

You can also clone the snappl archive from the `Github repo`_ and work with it there (e.g. using ``pip install -e .`` in your git checkout).

.. _Github repo: https://github.com/Roman-Supernova-PIT/snappl
